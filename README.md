# cplib

A competitive programming library by Helington Willamy.

## Structure

```
cplib/
├── code/
│   ├── data structures/
│   ├── dynamic programming/
│   ├── extra/
│   ├── geometry/
│   ├── graph/
│   ├── math/
│   └── string/
├── .gitignore
└── README.md
```

## Usage

Browse the `code/` directory and copy the implementation you need into your solution. Each file includes:
- Brief description
- Time complexity analysis
- Ready-to-use implementation

## Template

A standard competitive programming template is available in `code/extra/template.cpp` with:
- Fast I/O configuration
- Common macros and type definitions
- Multi-test case support
- Optional file I/O
